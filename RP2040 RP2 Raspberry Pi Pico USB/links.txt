Operation method for updating firmware:
1. First, connect the other end of the TYPE-C interface USB cable to the computer;
2. Press the BOOT button repeatedly;
3. Depress the RST key;
4. After one second, release the BOOT button;
5. It will be instantly identified by the computer as a detachable hard drive;
6. The detachable hard drive will appear; copy the file to be downloaded (burned) to it, and the core board will resume automatically.
